## Key Learning Areas from Reference Sites

### Theme Switching Excellence
**Sites to Study**: Vercel, Linear, Rauno, Figma, Stripe
- Seamless light/dark transitions (< 200ms)
- No flash or white flicker
- Respects system preference
- Persistent preference storage
- Smooth color animations

### Micro-Interactions & Animation
**Sites to Study**: Rauno, Daniel Eden, Josh Comeau, Tobias Ahlin, Shu Ding
- Spring easing curves for natural motion
- Reduced motion respect
- Staggered animations for depth
- Hover states with elevation
- Loading state feedback

### Grid-Based Layouts
**Sites to Study**: Linear, Figma, Vercel, Cursor
- 12-column grid systems
- Consistent spacing rhythm
- Responsive breakpoints
- Card component systems
- Mobile-first approach

### Card Component Design
**Sites to Study**: Linear, Figma, Stripe, Amie, Sunday
- Minimal borders with subtle shadows
- Hover elevation effects
- Color transitions on interaction
- Icon/image optimization
- Truncation and text overflow handling

### Typography Systems
**Sites to Study**: Rasmus Andersson, Niki Tonsky, Maxime Heckel, Josh Comeau
- Geometric sans-serif for UI (Inter, SF Pro)
- Serif fonts for articles and quotes
- Consistent line-height scaling
- Color contrast optimization
- Readable font sizes (14px minimum for body)

### Accessibility Practices
**Sites to Study**: Vercel, Design Engineer, Making Software, Amelia Wattenberger
- WCAG 2.1 AA compliance
- Visible focus indicators
- Keyboard navigation support
- Screen reader optimization
- Color contrast ratios

### Performance Optimization
**Sites to Study**: Linear, Vercel, Cursor, Poolside
- Fast page loads (LCP < 2.5s)
- Smooth scrolling (60fps)
- Lazy loading assets
- CSS-in-JS optimizations
- Bundle size consciousness
